los_memory.h
UINT32 LOS_MemLastUsedGet ( VOID *  pPool  ) 

Get the address of last node.

Description:
  • This API is used to get the address of last node.
注意:
  • The input pPool parameter must be initialized via func LOS_MemInit.
  • The last node of memory pool is not the end node.
参数:
参数名称 描述
pPtr  [IN] A pointer pointed to the memory pool.
返回值:
返回值 描述
LOS_NOK  The incoming parameter pPool is NULL.
UINT32  The address of the last used node that casts to UINT32.
Dependency:
  • los_memory.h: the header file that contains the API declaration.
参见:
None.
