atiny_net_recv
Agenttiny Socket
int atiny_net_recv ( void *  ctx,
unsigned char *  buf,
size_t  len  
)

read characters from peer.

Description:
This API is used to read at most 'len' characters. If no error occurs, the actual amount read is returned.
注意:
none.
参数:
参数名称 描述
ctx  [IN] Pointer to the socket.
buf  [IN] The buffer to write to.
len  [IN] Maximum length of the buffer.
返回值:
返回值 描述
int  the number of bytes received.
0  indicates recv() would block.
-1  recv failed, you should try to connect again.
Dependency: none.
参见:
atiny_net_connect | atiny_net_send | atiny_net_recv_timeout | atiny_net_close
