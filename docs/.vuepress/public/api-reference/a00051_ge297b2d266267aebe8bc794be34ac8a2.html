atiny_snprintf
Agenttiny Adapter
int atiny_snprintf ( char *  buf,
unsigned int  size,
const char *  format,
  ...  
)

writes formatted data to string.

Description:
This API is used to writes formatted data to string.
注意:
none.
参数:
参数名称 描述
buf  [OUT] string that holds written text.
size  [IN] maximum length of character will be written.
format  [IN] format that contains the text to be written, it can optionally contain embedded format specifiers that specifies how subsequent arguments are converted for output.
...  [IN] the variable argument list, for formatted and inserted in the resulting string replacing their respective specifiers.
返回值:
返回值 描述
int  bytes of character successfully written into string.
Dependency: none.
参见:
none.
