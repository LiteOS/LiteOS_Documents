LOS_IntLock
Hardware interrupt
UINTPTR LOS_IntLock ( VOID   ) 

Disable all interrupts.

Description:
  • This API is used to disable all IRQ and FIQ interrupts in the CPSR.
注意:
  • None.
参数:
参数名称 描述
None. 
返回值:
返回值 描述
CPSR  value obtained before all interrupts are disabled.
Dependency:
  • los_hwi.h: the header file that contains the API declaration.

LOS_IntRestore
